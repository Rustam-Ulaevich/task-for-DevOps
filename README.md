Условие задачи
Напиши на Go программу «Здоровье микросервисов».

На вход подается список URL-адресов (например, эндпоинты /health ваших микросервисов).
Программа должна:

Асинхронно (используя горутины) опросить все URL.

Для каждого URL проверить, что:

Статус ответа равен 200 OK.

Тело ответа содержит строку "ok" (регистр не важен).

Установить таймаут на каждый запрос — 2 секунды. Если сервер не ответил за 2 секунды — считать сервис недоступным.

Вывести в stdout таблицу результатов вида:

text
[OK]     https://service1/health
[FAIL]   https://service2/health (timeout)
[FAIL]   https://service3/health (status 500)
[OK]     https://service4/health
Программа должна завершиться не дольше, чем через 3 секунды от старта (общий таймаут выполнения).
